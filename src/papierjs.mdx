## papierjs

**papierjs** *(papier is paper in french)* provides web component built with stencil using 
papercss style.

### Installation

```bash
    npm install @amiceli/papierjs -S
```

Init components : 

```javascript
import { defineCustomElements } from '@amiceli/papierjs'

defineCustomElements()
```
