# p-leaf

`p-leaf` allowes to use **papercss** style/class in specific DOM element.

```html
<p-leaf>
    <span class="badge">123</span>
</p-leaf>
<p-leaf>
    <img 
        style="width: 200px"
        src="https://raw.githubusercontent.com/amiceli/papierjs/refs/heads/main/src/assets/papierjs.webp"
    />
</p-leaf>
```

<p-leaf>
    <span className="badge">123</span>
</p-leaf>
<p-leaf>
    <img 
        style={{width: "200px"}}
        src="https://raw.githubusercontent.com/amiceli/papierjs/refs/heads/main/src/assets/papierjs.webp"
    />
</p-leaf>

## Dark mode

```html
<!-- usefull for demo but dark property is enough ;) -->
<div style={{padding: '10px', boxSizing: "border-box", width: "100%", background: "#41403e"}}>
    <p-leaf dark>
        <input type="button" className="btn-secondary" value="Secondary"/>
    </p-leaf>
</div>
```
