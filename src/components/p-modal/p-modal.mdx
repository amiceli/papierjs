# p-modal

```html
<p-modal id="modal-1">
    <h1>Awesome</h1>
</p-modal>
<p-button 
    type="success"
    id="open-modal-1"
>Open modal</p-button>
```

```javascript
const modal = document.querySelector('#modal-1')
const buttonModal = document.querySelector('#open-modal-1')

buttonModal.addEventListener('click', () => {
    modal.open()
})
```

## Title, Sub title and Text

```html
<p-modal id="modal-2">
    <span slot="title">Modal title</span>
    <span slot="sub-title">Modal sub title</span>
    <span slot="text">This is an example of modal which is implemented with pure CSS! :D</span>
    <p-button type="warning">Close it !</p-button>
</p-modal>
```
