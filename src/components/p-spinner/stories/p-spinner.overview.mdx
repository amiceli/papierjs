# p-spinner

## color

You can pass `color` as props or spinner use current color : 

<p-spinner color="#eee"></p-spinner>

<p style={{color:"orange"}}>
    <p-spinner></p-spinner>
</p>

<Source language='html' dark format={true} code={`
    <p-spinner color="#eee"></p-spinner>

    <p style={{color:"orange"}}>
        <p-spinner></p-spinner>
    </p>
`} />

## Size

You can use a `large` p-spinner : 

<p-spinner large color="#eee"></p-spinner>

    <p-spinner large color="#eee"></p-spinner>
`} />
