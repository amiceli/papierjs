## onChange

`p-dropdown` listen `change` event on each `p-dropdown-item`. 

When catch this event,
`p-dropdown` will fire `select` event.

```javascript
const onChange = document.querySelector('#on-change-dropdown')
onChange.addEventListener('select', (e: CustomEvent) => {
    window.alert(e.detail)
})
```

### Placeholder

If any `p-dropdown-item` is `selected`, you can set a placeholder.

```html
<p-dropdown placeholder="Choose video game">
    <p-dropdown-item value="unity">AC Unity</p-dropdown-item>
    <p-dropdown-item value="origin">AC Origin</p-dropdown-item>
</p-dropdown>
```
